Q. Write program to show how to append, pop and sort elements in array in javascript.


Sorting:

let numbers = [0, 1 , 2, 3, 10, 20, 30 ];
numbers.sort();
console.log(numbers);

OUTPUT: [ 0, 1, 10, 2, 20, 3, 30 ]

Pop:

const fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.pop();

OUTPUT:[Banana,Orange,Apple]

Append :

const countries = ["India", "England", "Uganda"];
countries.push("Kenya");

OUTPUT : ["India", "England", "Uganda", "Kenya"]
